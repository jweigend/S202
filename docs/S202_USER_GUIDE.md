# S202 User Guide

## Goal

This document describes S202 from the user's perspective. It explains what the UI shows, how central terms are understood in the tool, and which typical steps a user performs.

## Planned Content

1. Overview of S202
2. Load a project or JAR
3. Read the layered representation
4. Understand packages, classes, and levels
5. Show and hide dependencies
6. Interpret violations
7. Perform what-if moves
8. Mark and check CUTs
9. Use results for refactoring planning
